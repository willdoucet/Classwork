Project Overview

The purpose of this project was to collect economic data and visualize it in a way that would paint a picture of how major economic measures have changed over time.

The project involved the use of:


  • Python - To pull and organize the data
  • SQLite - To store the data
  • JavaScript - To run scripts for the site
  • Plotly Library - To visualize the data
  • Flask - To create the web application

Grey vertical bars found in some of the graphs represent recession periods.
